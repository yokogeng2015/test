# 入口图自动生成工具

这是一个无需构建步骤的浏览器小工具，用于把一张主画面批量生成多个入口图尺寸。主画面可以包含文字、icon、人物或背景；你可以为每个尺寸上传 demo 图、调整安全区参数，并勾选需要输出的入口图尺寸。

## 功能

- 上传一张主画面图片并即时预览。
- 新增、删除、勾选任意入口图尺寸。
- 每个尺寸都可以上传一张 demo 图，并明确选择其用途：构图参考、背景/边框模板，或文字/icon/安全区标记。
- 每个尺寸都可以配置宽度、高度、主画面 X/Y/宽/高、圆角、留白、安全区 X/Y、背景色和生成方式。
- 支持“铺满裁切”和“完整留白”两种生成方式。
- 一键生成所有勾选尺寸，并支持单张下载或全部逐个下载。
- 尺寸配置会保存到浏览器 localStorage，下次打开自动恢复。

## 使用方式

1. 直接用浏览器打开 `index.html`。
2. 在“上传主画面”区域选择一张主画面图片。
3. 在“配置入口图尺寸”区域勾选需要生成的尺寸，或点击“新增尺寸”添加新规格。
4. 如需参考某个入口位效果，可为对应尺寸上传 demo 图，并选择 demo 用途。
5. 根据主画面内容调整“主画面区域设置”（X/Y/宽/高、圆角、留白）和安全区 X/Y，避免完全依赖不可解释的图片自动识别。
6. 点击“生成勾选尺寸”。
7. 在“生成结果”区域预览并下载图片。

## 默认尺寸

工具内置以下默认尺寸：

- 方形入口：300 × 300
- 横版 Banner：640 × 360
- 运营入口：750 × 420
- 竖版海报：1080 × 1920

你可以删除默认尺寸，也可以新增更多尺寸。点击“恢复默认尺寸”会清空当前自定义尺寸配置并恢复上述默认值。

## demo 图和生成规则

每张 demo 图都需要先明确用途，避免把“参考图”“模板图”和“标注图”混在一起使用：

1. **目标构图参考**：demo 只用于说明目标版式、构图比例和视觉方向。默认规则下，输出尺寸由当前尺寸配置决定，主画面按“主画面区域设置”绘制，demo 最多作为半透明参考层出现在预览中，不直接参与最终 PNG 输出。
2. **背景/边框模板**：demo 是最终图的一部分，例如固定背景、边框、装饰框或遮罩底图。生成时会先在 canvas 中绘制 demo 背景，再将主画面绘制到配置的主画面区域。
3. **文字、icon、安全区位置标记**：demo 只用于标记哪些区域应保留或避让，例如标题区、角标、按钮、logo 或安全区。它和构图参考一样不直接参与最终输出，可作为半透明参考层辅助预览。

推荐默认规则如下：

- **输出尺寸由配置决定**：最终 canvas 使用每个尺寸填写的宽度和高度，不从 demo 自动反推输出尺寸。
- **主画面区域可编辑**：每个尺寸都提供主画面 X/Y/宽/高、圆角、留白、背景色、安全区 X/Y 等参数，优先使用这些可解释配置，而不是完全依赖图片自动识别。
- **主画面等比裁切铺满**：默认使用“铺满裁切”，让主画面等比裁切并铺满主画面区域；也可切换为“完整留白”。
- **demo 默认不参与最终输出**：当 demo 用途为构图参考或标记图时，只在预览中以半透明参考层显示，不写入下载的最终 PNG。

生成方式说明：

- **铺满裁切**：主画面会铺满“主画面区域”，超出部分被裁切，适合运营入口图、Banner 等视觉图。
- **完整留白**：完整保留主画面，主画面区域内的空白由 canvas 背景色或模板底图承接，适合必须完整保留文字或 icon 的图片。
- **主画面区域 X/Y/宽/高**：控制主画面绘制到输出 canvas 的哪个矩形区域。模板图场景下，通常需要把这个区域对齐 demo 中的空槽或相框位置。
- **圆角与留白**：圆角会裁剪主画面区域边缘；留白会在主画面区域内部再缩进，便于给边框、标题或按钮预留空间。
- **安全区 X/Y**：用于控制裁切焦点。默认 50/50 表示居中；如果重要内容在左侧，可降低 X；如果重要内容在上方，可降低 Y。
- **背景色**：支持 `#ffffff` 这类 CSS 颜色值；填写 `transparent` 可输出透明背景。

> 注意：浏览器无法从任意 demo 图中可靠识别所有文字、icon 和版式语义。本工具提供可解释、可手动微调的参数化生成方式，适合快速批量出图和入口图初稿制作。

## 本地数据

尺寸配置保存在当前浏览器的 localStorage 中。上传的主画面和 demo 图只在当前页面会话中使用，不会上传到服务器，也不会长期保存。
