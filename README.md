# 入口图智能重排生成工具

这是一个无需构建步骤的浏览器小工具，用于把一张主画面生成多个尺寸的入口图。新版重点修正为：**主画面可以本身就包含文字、icon、logo、角标等设计元素**，工具会尝试从主图中识别这些视觉块，再按照每个尺寸上传的 demo 图提示的位置、大小和比例重新排版生成新图片，而不是只做裁切。

## 核心能力

- 上传一张已经设计好的主画面，主画面可以包含文字、icon、logo、按钮或角标。
- 自动分析主图中的高对比/边缘视觉块，并把它们映射为 logo、主标题、副标题、角标等可搬运区域。
- 每个入口图尺寸都可以上传一张 demo 图，demo 用来提示该尺寸中类似文字、logo、角标应该放在哪里、占多大。
- 生成时默认使用“从主图自动提取图层”模式：保留主图原有文字/icon外观，将这些图层裁出后缩放、移动到 demo 推断出的区域。
- 如果主图自动识别不准确，可切换到“手动文案 / logo 图层”模式，手动输入主标题、副标题、角标和 logo。
- 新增、删除、勾选任意入口图尺寸。
- 支持背景“铺满裁切”“完整保留”“模糊铺底 + 完整主体”三种处理方式；自动提取模式会生成模糊背景，降低原主图中文字重复出现的概率。
- 一键生成所有勾选尺寸，并支持单张下载或全部逐个下载。
- 尺寸配置会保存到浏览器 localStorage，下次打开自动恢复。

## 使用方式

1. 直接用浏览器打开 `index.html`。
2. 上传一张主画面图片。上传后，下方会显示主图分析结果，包含识别到的 logo、标题、副标题、角标区域。
3. 在“内容来源”中选择：
   - **从主图自动提取图层（推荐）**：适合主画面已经包含文字、logo、icon 的情况。
   - **使用下面手动文案 / logo 图层**：适合想重新输入文字、替换 logo 的情况。
4. 在“配置入口图尺寸和 demo”区域勾选需要生成的尺寸，或点击“新增尺寸”添加新规格。
5. 为每个尺寸上传对应 demo 图。demo 会提示该尺寸里类似文字、logo、按钮/角标应出现的位置和大小。
6. 如某个尺寸没有 demo，工具会使用默认安全布局。
7. 点击“按主图内容 + demo 布局生成”。
8. 在“生成结果”区域预览并下载图片。

## 默认尺寸

工具内置以下默认尺寸：

- 方形入口：300 × 300
- 横版 Banner：640 × 360
- 运营入口：750 × 420
- 竖版海报：1080 × 1920

你可以删除默认尺寸，也可以新增更多尺寸。点击“恢复默认尺寸”会清空当前自定义尺寸配置并恢复上述默认值。

## demo 图如何影响生成

demo 图主要用于提供版式提示，而不是直接覆盖到最终图片中。

工具会在 demo 中寻找边缘、高对比和高色彩差异的内容区域，并将这些区域归类为：

- logo / icon 区域
- 主标题区域
- 副标题区域
- 角标 / CTA 区域

自动提取模式下，工具会从主图中找到对应视觉块，把主图里的原始文字/icon/logo裁出，再放到 demo 推断出的目标区域。因此输出结果是重新排版的新图，不再只是把主图按目标尺寸裁切。

## 当前限制

浏览器端没有 AI 抠图、OCR 或 PSD 图层信息，无法百分百理解任意设计图的语义。因此自动提取采用图像边缘和视觉块检测：

- 主图或 demo 背景过于复杂时，可能把背景纹理误识别为文字块。
- 如果主图中的文字和背景对比太低，可能漏识别。
- 自动提取模式会使用模糊背景来减少原文字重复，但不能像专业修图工具一样完美去除原图上的文字。
- 如果需要绝对可控的文字内容和字号，请切换为手动文案图层模式。

## 本地数据

尺寸配置保存在当前浏览器的 localStorage 中。上传的主画面、logo 和 demo 图只在当前页面会话中使用，不会上传到服务器，也不会长期保存。
