# 入口图智能重排生成工具

这是一个无需构建步骤的浏览器小工具，用于把一张主画面生成多个尺寸的入口图。新版不再只是裁切图片，而是把主画面作为背景素材，并把标题、副标题、logo、角标等作为可重新排版的图层绘制到新图片中。

## 核心能力

- 上传一张主画面作为背景素材。
- 输入主标题、副标题、角标文案，并配置文字颜色、强调色、字体、阴影和文字区域暗底。
- 上传 logo / icon 图片；如果不上传，会使用文字 logo 占位。
- 新增、删除、勾选任意入口图尺寸。
- 每个尺寸都可以上传 demo 图作为布局参考。
- 工具会分析 demo 中的高对比内容块，推断 logo、主标题、副标题、角标大概排在哪里、占多大，并按该布局重新生成图片。
- 支持背景“铺满裁切”“完整保留”“模糊铺底 + 完整主体”三种处理方式。
- 一键生成所有勾选尺寸，并支持单张下载或全部逐个下载。
- 尺寸配置会保存到浏览器 localStorage，下次打开自动恢复。

## 使用方式

1. 直接用浏览器打开 `index.html`。
2. 在“上传主画面背景”区域选择一张主画面图片。
3. 在“设置可重排内容图层”区域填写主标题、副标题、角标，并按需上传 logo / icon。
4. 在“配置入口图尺寸和 demo”区域勾选需要生成的尺寸，或点击“新增尺寸”添加新规格。
5. 为每个尺寸上传对应 demo 图。demo 会提示该尺寸里类似文字、logo、按钮/角标应出现的位置和大小。
6. 如某个尺寸没有 demo，工具会使用默认安全布局。
7. 点击“按 demo 重排生成”。
8. 在“生成结果”区域预览并下载图片。

## 默认尺寸

工具内置以下默认尺寸：

- 方形入口：300 × 300
- 横版 Banner：640 × 360
- 运营入口：750 × 420
- 竖版海报：1080 × 1920

你可以删除默认尺寸，也可以新增更多尺寸。点击“恢复默认尺寸”会清空当前自定义尺寸配置并恢复上述默认值。

## demo 图如何影响生成

demo 图主要用于提供版式提示，而不是直接覆盖到最终图片中。

工具会在 demo 中寻找与背景差异较大的内容区域，并将这些区域归类为：

- logo / icon 区域
- 主标题区域
- 副标题区域
- 角标 / CTA 区域

生成时，工具会把你输入的文案和上传的 logo 按这些区域的位置、尺寸重新绘制到目标尺寸中。因此输出结果是新的入口图，而不是把主图简单裁切成不同大小。

## 背景和图层规则

- **背景铺满裁切**：主画面铺满目标尺寸，适合横版 Banner 和运营入口图。
- **完整保留背景**：完整保留主画面，空白区域使用背景色填充。
- **模糊铺底 + 完整主体**：先用主画面模糊铺满背景，再把完整主画面放在中间，适合比例差异较大的尺寸。
- **文字图层**：根据 demo 推断出的标题/副标题区域自动换行和缩放。
- **logo 图层**：优先使用上传的 logo / icon；未上传时使用文字 logo 占位。
- **角标图层**：使用强调色绘制圆角按钮或角标。

## 限制说明

浏览器端不能可靠地从任意图片中自动 OCR 出文字内容，也不能准确理解所有设计语义。因此该工具采用“可编辑内容图层 + demo 自动布局识别”的方式：

- 文案需要用户输入，保证最终文字清晰可控。
- demo 用来推断位置、大小和排列，而不是识别文案内容。
- 如 demo 背景和内容对比太低，识别可能不准确；这种情况下会回退到默认安全布局。

## 本地数据

尺寸配置保存在当前浏览器的 localStorage 中。上传的主画面、logo 和 demo 图只在当前页面会话中使用，不会上传到服务器，也不会长期保存。
