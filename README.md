# 入口图自动生成工具

这是一个无需构建步骤的浏览器小工具，用于把一张主画面批量生成多个入口图尺寸。主画面可以包含文字、icon、人物或背景；你可以为每个尺寸上传 demo 图、调整安全区参数，并勾选需要输出的入口图尺寸。

## 功能

- 上传一张主画面图片并即时预览。
- 新增、删除、勾选任意入口图尺寸。
- 每个尺寸都可以上传一张 demo 图作为构图参考。
- 每个尺寸都可以配置宽度、高度、安全区 X/Y、背景色和生成方式。
- 支持“铺满裁切”和“完整留白”两种生成方式。
- 一键生成所有勾选尺寸，并支持单张下载或全部逐个下载。
- 尺寸配置会保存到浏览器 localStorage，下次打开自动恢复。

## 使用方式

1. 直接用浏览器打开 `index.html`。
2. 在“上传主画面”区域选择一张主画面图片。
3. 在“配置入口图尺寸”区域勾选需要生成的尺寸，或点击“新增尺寸”添加新规格。
4. 如需参考某个入口位效果，可为对应尺寸上传 demo 图。
5. 根据主画面内容调整安全区 X/Y，避免文字或 icon 被裁切。
6. 点击“生成勾选尺寸”。
7. 在“生成结果”区域预览并下载图片。

## 默认尺寸

工具内置以下默认尺寸：

- 方形入口：300 × 300
- 横版 Banner：640 × 360
- 运营入口：750 × 420
- 竖版海报：1080 × 1920

你可以删除默认尺寸，也可以新增更多尺寸。点击“恢复默认尺寸”会清空当前自定义尺寸配置并恢复上述默认值。

## demo 图和生成规则

目前 demo 图主要作为“构图参考”使用：工具会读取 demo 的比例，并结合目标尺寸、安全区 X/Y、生成方式推断主画面的裁切位置。

- **铺满裁切**：主画面会铺满目标尺寸，超出部分被裁切，适合运营入口图、Banner 等视觉图。
- **完整留白**：完整保留主画面，空白区域使用配置的背景色填充，适合必须完整保留文字或 icon 的图片。
- **安全区 X/Y**：用于控制裁切焦点。默认 50/50 表示居中；如果重要内容在左侧，可降低 X；如果重要内容在上方，可降低 Y。
- **背景色**：支持 `#ffffff` 这类 CSS 颜色值；填写 `transparent` 可输出透明背景。

> 注意：浏览器无法从任意 demo 图中可靠识别所有文字、icon 和版式语义。本工具提供可解释、可手动微调的自动生成方式，适合快速批量出图和入口图初稿制作。

## 本地数据

尺寸配置保存在当前浏览器的 localStorage 中。上传的主画面和 demo 图只在当前页面会话中使用，不会上传到服务器，也不会长期保存。
